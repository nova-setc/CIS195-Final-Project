Stardew Valley Fanpage

This project was designed to test our knowledge of html elements such as tables, forms, and embedded media, as well as our ability to design them responsively. We were allowed to select any topic for this project, so long as the requirements were met. I chose to design a website focused on providing information about one of my favorite video games: Stardew Valley.

You can visit the site here.

Responsive Columns

One of the major things we focused on this semester was responsive web design, with a focus on mobile-first strategies. Here you can see an example of a column-based layout which automatically adjusts to a single column on smaller screens. This allows for ease of visibility to mobile users, while still keeping the website condensed and pleasing for desktop users. We learned multiple ways to create column layouts such as CSS Grids, Bootstrap, and Float.

Forms

One of the primary focuses of this project was on creating web forms. This form collects users' opinions using radio buttons, checkboxes, and text-boxes designed to accept different kinds of input. The form is responsive and adjusts based on the size of the user's screen.

Responsive Navigation

Here is another example of responsive web design. The navigation bar is designed to be horizontal to take up less space when the user is viewing the site on a large screen. The navigation switches to a vertical layout on smaller screens for increased legibility.
