The Future of Robotics

This project was designed to increase our knowledge of mobile-first strategies and creating responsive website designs. Our topic was future technologies, and I chose to create a website focuses on the future of advanced robotics.

You can visit the site here.

Responsive Layout

The primary focus of this project was showcasing what we had learned about mobile-first responsive web design. Here you can see the layout and imagery adjusting based on the size of the screen.

Timeline

While not a requirement of this project, I decided that I wanted to use CSS to style a timeline for this project. I felt that this was most in line with my initial vision for the website. I had to learn about the pseudo classes :before and :after in order to be able to appropriately modify the desktop-first example I found to a mobile-first stylesheet.

Responsive Figures

The figures on this website were also designed to be responsive. On large screens the text and image are grouped together with a background signaling that they are part of one unit. On smaller screens, the size of the text for the headings is used to indicate a new section of text.
